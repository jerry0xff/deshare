# 合约需求整理

## 1. 合约概述
- **功能**：EVM合约，支撑整个交易环节

## 2. 管理员功能
- 可设置手续费接收地址
- 可设置交易资金接收地址
- 可设置股票凭证代币的接收地址

## 3. 股票买入交易流程
### 3.1 买单流程
1. 用户调用合约进行交易，输入金额、股票名称、交易类型（限价或市价，买/卖）
2. 合约收取订单总金额USDC和手续费
3. 将资金分别发往对应的账户
4. 后端根据事件购买真实股票
5. 后端购买完成后，使用管理员签名调用合约生成代币凭证

## 4. 股票卖出交易流程
### 4.1 卖单流程
1. 用户调用合约进行交易，输入股票数量、股票名称、交易类型（限价或市价，买/卖）
2. 合约收取订单的股票凭证代币和手续费
3. 将资金分别发往对应的账户
4. 在链上生成事件
5. 后端根据事件卖出真实股票
6. 后端卖出完成后，使用管理员签名调用合约：
   - 销毁代币凭证
   - 发送卖出得到的USDC给用户
